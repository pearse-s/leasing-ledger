# Leasing Ledger

Static front-end for the Live Furnished Leasing Ledger dashboard.
Data comes from the password-gated Supabase edge function `dashboard` (project yplsskzukmnvdnkzlygt).
This repo holds only the app shell (no data, no secrets) and is served via GitHub Pages.

Update flow: regenerated from `supabase/functions/dashboard/html.ts` in the SupaBase project, committed here, auto-deploys on push.
